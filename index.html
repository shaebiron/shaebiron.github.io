How to Create Your Personal Webpage on Github

It's quick and simple using these easy steps:

  • Sign up for a Github account.
  • Click “Create a repository” and create a repository name (it should be your .github.io)
  • Make it Public and Add a README file by clicking the boxes next to these options. Plus, Choose a license (select License: Apache License 2.0 from the pull-down).
  • Click Create repository button.
